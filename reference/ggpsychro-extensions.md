# ggpsychro extensions to ggplot2

ggpsychro makes heavy use of the ggproto class system to extend the
functionality of ggplot2. In general the actual classes should be of
little interest to users as the standard ggplot2 api of using geom\_\*
and stat\_\* functions for building up the plot is encouraged.

## Value

A ggproto object used internally by ggpsychro extensions.
